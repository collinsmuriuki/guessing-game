<h1 align="center">Rust guessing-game</h1>

A simple program that covers the fundamentals of Rust:
* Variables
* Ownership
* Modules
* Structs
* Functions
* Error handling
* Crates
* Testing

## Usage

```sh
git clone https://github.com/collinsmuriuki/guessing-game.git
cd guessing-game
cargo run
```

## Test
```sh
cargo test
```
